AddOn for Plugin WP Content Security Policy to dequeue Third Party jQuery-UI stylesheet and enqueue same stylesheet from own server.

== Description ==

Get rid of third party CSS and enqueue own to comply with GDPR.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= A question that someone might have =

Is this the optimal solution?

No. Instead the plugin author should be informed, that enqueueing third party content is a violation of the GDPR.
